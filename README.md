# nestjs-kit

[NestJS](https://nestjs.com) ([fastify](https://fastify.dev) + [socket.io](https://www.npmjs.com/package/socket.io)) kit.

## Sub Modules

-   `zod` - Zod pipes
-   `fastify-multipart` - Multipart support for fastify
-   `client` - Some client utilities to deal with responses

## Client

Use the `nestjs-kit-client` package.
